Freelance projects during the period between late 2016 and early 2018

Convocados

Collaborated with a backend developer, a product manager and a design/UX team in an agile environment to build a React Native application for both Android and iOS.

In addition to building the mobile app, I took part in shaping up and refining new features and ideas, and helped the backend developer conceptualise and structure the different endpoints for the RESTful API.

In Search of Snow

Collaborated with a backend developer and a team of UX and visual designers to ship a mobile-first SPA built using React and Redux.

Loan Calculator

Bespoke loan calculator built for a financial institution. This tool can be accessed by clients on their online account and helps them determine the monthly payments on a loan.

Users simply need to enter the loan amount, term and interest rate in the fields and the tool will provide all the details. They can also download and print out reports generated by the app.

This application was built from scratch using React. The standalone version is published to the client's site, but there's also a WordPress widget integration that allows the client to embed it onto their institutional site.

I was given a few examples of different types of loans with different values, and had to work on implementing the math to perform the calculations. I was the only developer on this project, and there were no designers or people from other disciplines involved.

Smartline

Led the rewrite from scratch of an online news platform for a local broadcasting company.

The project consisted of a RESTful API built using Node/Express/Sequelize/MySQL, a control panel to manage content and a server-side rendered single-page application, both powered by React and Redux.

In addition to that, I also developed a TV slider + news ticker using React and Bacon.js, which connected to the RESTful API mentioned above. This full-screen app goes live all day while there's no other content being broadcasted.

The API runs on DigitalOcean, the website, control panel and slider are hosted on Netlify, all media gets hosted on DigitalOcean Spaces, emails are sent by Mailgun, and all the automation (CI/CD) is managed by CircleCI.
